#  Aplicación Web Fórmula 1

Proyecto académico de **Ingeniería de Sistemas** enfocado en el desarrollo de una aplicación web sobre la **Fórmula 1**, que integra frontend y backend mediante una arquitectura **full stack**. El sistema permite consultar información relacionada con pilotos y equipos, así como registrar visitantes, simulando un entorno real de desarrollo de software.

---

##  Descripción del Proyecto

La aplicación fue desarrollada como parte de proyectos académicos con el objetivo de aplicar conceptos de:

* Desarrollo web moderno
* Arquitectura cliente-servidor
* APIs REST
* Organización del código en capas

El proyecto está orientado a demostrar habilidades prácticas en **React**, **Node.js** y **Express**, más allá de la teoría.

---

##  Tecnologías Utilizadas

### Frontend

* React
* HTML5
* CSS3
* JavaScript (ES6+)

### Backend

* Node.js
* Express
* API REST

### Otras herramientas

* Git y GitHub
* Visual Studio Code

---

##  Funcionalidades Principales

*  Página de inicio con información general de la Fórmula 1
*  Consulta de pilotos
*  Consulta de equipos
*  Sección de historia de la Fórmula 1
*  Registro de visitantes con los siguientes datos:

  * Nombre completo
  * Edad
  * Equipo favorito
  * Piloto favorito
*  Comunicación frontend-backend mediante solicitudes HTTP
*  Estructura modular con rutas y controladores en el backend

---

##  Arquitectura del Proyecto

El proyecto sigue una estructura similar a entornos reales:

* **Frontend:** Manejo de vistas, componentes y formularios en React.
* **Backend:**

  * Rutas (`routes`)
  * Controladores (`controllers`)
  * Lógica de negocio separada

Esta organización facilita el mantenimiento, la escalabilidad y la comprensión del sistema.

---

##  Objetivo Académico

El objetivo principal de este proyecto es:

* Aplicar conocimientos de desarrollo web full stack
* Comprender el flujo de datos entre cliente y servidor
* Simular un proyecto real de software
* Fortalecer habilidades en diseño de APIs REST
